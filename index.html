How to See What Kind of Processor You Have (ARM, ARM64, or x86)

1. Go to 'Settings'

2. Go to 'System' / 'About Phone' as the case may be.

3. Check 'Kernel Version'

4. If the code inside contains 'x64' string, your device runs a 64-bit OS.

5. Almost all the phones use ARM nowadays. Most probably you will have to click one of the first 2 buttons given above.
